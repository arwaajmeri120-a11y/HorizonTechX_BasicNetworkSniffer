# Horizon TechX - Basic Network Sniffer

## Project Overview
This project is a Basic Network Sniffer developed in Python using the Scapy library. It captures live network packets and displays Source IP, Destination IP, and Protocol information.

## Features
- Captures live network packets
- Displays Source IP Address
- Displays Destination IP Address
- Detects TCP, UDP, and ICMP protocols

## Technologies Used
- Python
- Scapy
- Visual Studio Code
- Npcap

## Installation
1. Install Python
2. Install Scapy using:
pip install scapy

## Run
python network_sniffer.py
